# admin-vue3-element3

```bash
# 装依赖
yarn
# 本地运行
yarn dev
# 打包
yarn build
```
## 项目说明

- 本项目是基于 Vue-Element-Admin 样式的 Vue3 版本重构，省略了 Mock 与 测试代码
- 主用技术栈：Vue3 Element-Plus TypeScript
- 此版本为基础 demo, 后续会继续完善
